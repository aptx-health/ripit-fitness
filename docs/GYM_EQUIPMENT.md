# Iron Works Boulder - Equipment Catalog

Reference for exercise curation and workout generation. Exercises should map to equipment actually available at the gym.

## Layout & Proximity Zones

Equipment is spread across four rooms. Proximity matters for supersets and circuits:

- **Freeweight Room**: Squat rack, cables, freeweights, smith machine — all close together
- **Machine Room**: All resistance machines in one area
- **Functional Room**: Dual cable, open floor, kettlebells, dumbbells — good for circuits
- **Stretching Area**: Bands, mats, blocks
- **Cardio**: Treadmill, elliptical, bikes, rower

## Home/Minimal Equipment Set

For training outside the gym, exercises should be curated for:

- Bodyweight only
- Resistance bands
- Single pair of dumbbells (optional)

## Notes for Exercise Curation

- **Skip**: Chest fly machine exercises — equipment is poorly designed
- **Proximity-aware supersets**: Freeweight room gear is close together; machine room is separate. Don't program supersets that require running between rooms.
- **Duplicate equipment**: Cables, TRX, dumbbells, and benches exist in both freeweight and functional rooms
